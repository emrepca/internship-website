Welcome To Mono Bilisim

Mono Bilisim Incorporated Company was founded on February 4, 2010 in Istanbul. Mono specializes in free software solutions. Develops information and communication technologies using free software; provides consulting, training, system management and software development services.

Consultancy

Mono advises you on how to avoid vendor lock-in dependency in terms of both software and hardware in your projects; to produce alternatives to proprietary software with free software.

Software Development


Software Development for Asterisk

Using Asterisk Gateway Interface (AGI), Asterisk Manager Interface (AMI) and Asterisk REST Interface (ARI), we provide software development services for robot operators, voice response system, automatic call service, CRM integration, wallboard customizations.

Developing Web Applications
